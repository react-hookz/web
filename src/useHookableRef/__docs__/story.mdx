# useHookableRef

Like `React.useRef` but it is possible to define get and set handlers.

#### Example

<Canvas>
  <Story story={Example} inline />
</Canvas>

## Reference

```ts
export type HookableRefHandler<T> = (v: T) => T;

export function useHookableRef<T>(
  initialValue: T,
  onSet?: HookableRefHandler<T>,
  onGet?: HookableRefHandler<T>
): React.MutableRefObject<T>;
export function useHookableRef<T = undefined>(): React.MutableRefObject<T | undefined>;
```

#### Importing

<ImportPath />

#### Arguments

- **initialValue** _`any`_ Initial value of a hook.
- **onSet** _`HookableRefHandler`_ Function to be called while `ref.current` value set. Return value will be stored in ref.
- **onGet** _`HookableRefHandler`_ Function to be called while `ref.current` value accessed. Return value will be used as a return value.

#### Return

- _`React.MutableRefObject`_
