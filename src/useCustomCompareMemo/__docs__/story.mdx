# useCustomCompareMemo

Like useMemo but uses provided comparator function to validate dependency changes.

#### Example

## Reference

```ts
export type DependenciesComparator<Deps extends DependencyList = DependencyList> = (
	a: Deps,
	b: Deps
) => boolean;

function useCustomCompareMemo<T, Deps extends DependencyList>(
	factory: () => T,
	deps: Deps,
	comparator: DependenciesComparator<Deps>
): T;
```

#### Importing

#### Arguments

- **factory** _`() => unknown`_ - useMemo factory function.
- **deps** _`DependencyList`_ - useMemo dependency list.
- **comparator** _`DependenciesComparator<DependencyList>`_ - comparator function to validate dependency changes.
