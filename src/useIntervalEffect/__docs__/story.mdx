# useInterval

Like `setInterval` but in form of react hook.

- Auto-cancel interval on component unmount.
- Reset interval on delay change.
- Callback change dont cause interval reset.
- Ability to disable interval.

## Reference

```ts
export function useIntervalEffect(callback: () => void, ms?: number): void;
```

#### Importing

<ImportPath />

#### Arguments

- **callback** _`() => void`_ - function to call within interval.
- **ms** _`number | undefined`_ - delay passed to underlying `setInterval`. In case delay
  is `undefined` - interval will be cancelled.
