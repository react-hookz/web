# useDebouncedEffect

Like `useEffect`, but the passed function is debounced.

#### Example

## Reference

```ts
export function useDebouncedEffect(
	callback: (...args: any[]) => void,
	deps: DependencyList,
	delay: number,
	maxWait = 0
): void;
```

#### Importing

#### Arguments

- **callback** _`(...args: any[]) => void`_ - Callback like for `useEffect`, but without ability to
  return a cleanup function.
- **deps** _`DependencyList`_ - Dependency list like the one passed to `useEffect`.
- **delay** _`number`_ - Debounce delay (in milliseconds).
- **maxWait** _`number`_ _(default: `0`)_ The maximum time `callback` is allowed to be delayed
  before it's invoked. `0` means no max wait.
