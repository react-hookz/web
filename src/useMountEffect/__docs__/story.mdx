# useMountEffect

Run effect only when component is first mounted.

> This hook does not provide any cleanup functionality, meaning that effect's return not passed to
> underlying `useEffect`.

## Reference

```ts
function useMountEffect(effect: CallableFunction): void;
```

<ImportPath />

#### Arguments

- _**effect**_ _`CallableFunction`_ - Effector to call when component mounted. Can be a function
  returning any value. In case the need of cleanup action - use common `useEffect` with empty deps
  array.
