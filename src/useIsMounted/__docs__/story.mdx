# useMountedGetter

Returns function that yields current mount state. This hook is handy for the cases when you have to
detect component mount state within async effects.

> **_This hook provides stable API, meaning returned methods does not change between renders_**

## Reference

```ts
function useIsMounted(initialValue = false): () => boolean;
```

#### Importing

<ImportPath />

#### Arguments

- **initialState** _`boolean`_ _(default: `false`)_ - Initial value. By default, this hook
  assumes that hook is not mounted yet at first render.

#### Return

Function that yields `true` only in case component is mounted.
