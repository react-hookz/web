# useIsMounted

Returns a function that returns the current mount state. This hook is useful when you have to detect
component mount state within async effects.

> **_This hook provides stable API, meaning returned methods does not change between renders_**

#### Example

## Reference

```ts
function useIsMounted(initialValue = false): () => boolean;
```

#### Importing

#### Arguments

- **initialState** _`boolean`_ _(default: `false`)_ - Initial value.

#### Return

Function that returns `true` only if the component is mounted.
