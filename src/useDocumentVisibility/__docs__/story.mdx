# useDocumentVisibility

Returns a boolean indicating whether the document is visible or not.

#### Example

## Reference

```ts
function useDocumentVisibility(initializeWithValue = true): boolean | undefined;
```

#### Importing

#### Arguments

- **initializeWithValue** _`boolean | undefined`_ _(default: `true`)_ - Whether to initialize state
  with the document visibility state or `undefined`. _Set this to `false` during SSR._
