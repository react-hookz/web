# useSet

Tracks the state of a `Set`.

- Returned set does not change between renders.
- 1-to-1 signature with the native `Set` object, but its methods are wrapped to cause components to
  rerender with changes.  
  Otherwise, it is the Javascript native `Set` object.
- SSR-friendly.
