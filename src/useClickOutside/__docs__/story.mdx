# useClickOutside

Triggers a callback when the user clicks outside a targeted element.

> This hook listens for events on the `document` object. Therefore, if any listener between the event
> target and the `document` object stops event propagation, the hook won't work.

> By default, the `mousedown` and `touchstart` events are used, but any bubbling event can be used instead.

#### Example

## Reference

```ts
const DEFAULT_EVENTS = ['mousedown', 'touchstart'];

export function useClickOutside<T extends HTMLElement>(
	ref: RefObject<T> | MutableRefObject<T>,
	callback: EventListener,
	events: string[] = DEFAULT_EVENTS
): void;
```

#### Importing

#### Arguments

- **ref** _`RefObject<T> | MutableRefObject<T>`_ - React ref object containing the target element.
- **callback** _`EventListener`_ - Callback invoked when the user clicks outside the target element.
- **events**: _`string[]`_ _(default: `['mousedown', 'touchstart']`)_ - List of events that will be
  used as triggers for the outside click.
