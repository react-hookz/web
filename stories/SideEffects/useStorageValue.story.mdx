# useStorageValue

Manages a single storage key. General-purpose hook for string key-value storages, that used
in [`useLocalStorageValue`](/docs/side-effects-uselocalstoragevalue--example) and
[`useSessionStorageValue`](/docs/side-effects-usesessionstoragevalue--example).

> This hook requires storage to be synchronous, string-value storage.
