# useMountEffect

Run effect only when component is first mounted.

#### Example

<Canvas>
  <Story story={Example} />
</Canvas>

## Reference

```ts
function useMountEffect(effect: CallableFunction): void;
```

#### Arguments

- _**effect**_ _`CallableFunction`_ - Effector to call when component mounted. Can be a function returning any value.
  In case the need of cleanup action - use common `useEffect` with empty deps array.
