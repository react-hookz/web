# useToggle

Like `useState`, but can only become `true` or `false`.

#### Example

<Canvas>
  <Story story={Example} inline />
</Canvas>

## Reference

```ts
function useToggle(
  initialState: IInitialState<boolean> = false
): [boolean, (nextState?: INewState<boolean>) => void];
```

#### Arguments

- _**initialState**_ _`IInitialState<boolean>`_ - initial state or initial state setter as for `useState`

#### Return

Returns array alike `useState` does.

- _**[0]**_ _`boolean`_ - current state
- _**[1]**_ _`(nextState?: INewState<boolean>) => void`_ - state setter as for `useState`.
  In case called without arguments will change state to opposite.
